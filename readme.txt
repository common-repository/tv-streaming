== Description ==

TV Streaming is a WordPress plugin that can integrate live video streaming from various TV channels into your WordPress site. TV Streaming can be displayed on the page, post or on the sidebar of your site as a widget. For this first release version, TV Streaming plugin currently support all TV channels in Indonesia, such as: TransTV, Indosiar, SCTV, ANTV, GlobalTV, TVOne, MetroTV, etc.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `tv-streaming` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
